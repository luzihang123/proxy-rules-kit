# 客户端矩阵

| 客户端 | 推荐入口 | 自动地区匹配 | 适用场景 |
| --- | --- | --- | --- |
| Clash Verge Rev | `generic-script.js` | 是 | 桌面端主力，脚本注入规则和组 |
| Clash Party | `override.yaml` | 是 | Mihomo 覆写配置 |
| OpenClash | 三件套配置 | 是 | 路由器全局覆盖 |
| Shadowrocket | `routekit.conf` | 部分支持 | iOS 移动端 |
| Clash for Windows | `parsers.example.yaml` | 否 | 旧客户端兼容 |
